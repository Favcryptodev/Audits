Key Findings

High
H-01

Potential Reentrancy Vulnerability

The `unstake()` function updates the staking information before transferring the tokens back to the user. This could potentially allow a malicious user to repeatedly call the `unstake()` function before the state is updated, leading to a reentrancy attack.
LOCATION
Line 194-208
IMPACT
An attacker could repeatedly call the `unstake()` function to drain the contract's token balance.
RECOMMENDATION
Implement the Checks-Effects-Interactions pattern by updating the staking information before transferring the tokens. Additionally, consider using the OpenZeppelin ReentrancyGuard contract to protect against reentrancy attacks.
REVISION COMMENT
This is actually negated through an additional reentrancy guard implementation as well as manual management of reward balances, and requires reward balance fallback.
Medium
M-01

Potential Integer Overflow/Underflow

The contract uses the `SafeMath` library to perform arithmetic operations, but there are still some instances where integer overflow/underflow could occur. For example, in the `calculatePeriodRewards()` function, the `timeElapsed` variable is divided by 365 days, which could lead to an underflow if the `timeElapsed` is less than 365 days.
LOCATION
Line 311-321
IMPACT
An attacker could potentially exploit integer overflow/underflow vulnerabilities to manipulate the reward calculations.
RECOMMENDATION
Ensure that all arithmetic operations are properly bounded and use the `SafeMath` library consistently throughout the contract.
REVISION COMMENT
Safe math is depreciated

    /**
     * @dev Stake GRANDPEPE tokens
     * @param _amount Amount of tokens to stake
     */
    function stake(uint256 _amount) external nonReentrant {
        require(_amount > 0, "Cannot stake 0 tokens");
        
        // Update unclaimed rewards before modifying stake
        if (stakingInfo[msg.sender].amount > 0) {
            _updateUnclaimedRewards(msg.sender);
        }
        
        // Transfer tokens to contract
        IERC20(grandPepeToken).safeTransferFrom(msg.sender, address(this), _amount);
        
        // If first time staking, initialize staking info
        if (stakingInfo[msg.sender].startTime == 0) {
            stakingInfo[msg.sender] = StakingInfo({
                amount: _amount,
                startTime: block.timestamp,
                lastClaimTime: block.timestamp,
                totalClaimed: 0,
                unclaimedRewards: 0
            });
            numberOfStakers = numberOfStakers.add(1);
        } else {
            // Add to existing stake
            stakingInfo[msg.sender].amount = stakingInfo[msg.sender].amount.add(_amount);
            stakingInfo[msg.sender].lastClaimTime = block.timestamp;
        }
        
        // Update total staked amount
        totalStaked = totalStaked.add(_amount);
        
        emit Staked(msg.sender, _amount);
    }
    
    /**
     * @dev Unstake GRANDPEPE tokens
     * @param _amount Amount of tokens to unstake
     */
    function unstake(uint256 _amount) external nonReentrant {
        StakingInfo storage info = stakingInfo[msg.sender];
        require(info.amount >= _amount, "Insufficient staked amount");
        
        // Update unclaimed rewards before modifying stake
        _updateUnclaimedRewards(msg.sender);
        
        // Check if user is completely unstaking
        bool isFullyUnstaking = (info.amount == _amount);
        
        // Update staking info
        info.amount = info.amount.sub(_amount);
        
        // If user completely unstaked, decrement staker count
        if (isFullyUnstaking) {
            numberOfStakers = numberOfStakers.sub(1);
        }
        
        // Update total staked amount
        totalStaked = totalStaked.sub(_amount);
        
        // Transfer tokens back to user
        IERC20(grandPepeToken).safeTransfer(msg.sender, _amount);
        
        emit Unstaked(msg.sender, _amount);
    }

    /**
     * @dev Calculate rewards for a period with specific APR rates
     * This is a simplified version to avoid stack too deep errors
     * @param _amount Amount of tokens staked
     * @param _timeElapsed Time elapsed in seconds
     * @param _baseAPR Base APR for the period
     * @param _bonusAPR Bonus APR for the period
     * @param _bonusEligibleAmount Amount eligible for bonus APR
     * @return Rewards accrued for the period
     */
    function calculatePeriodRewards(
        uint256 _amount,
        uint256 _timeElapsed,
        uint256 _baseAPR,
        uint256 _bonusAPR,
        uint256 _bonusEligibleAmount
    ) internal pure returns (uint256) {
        if (_amount == 0 || _timeElapsed == 0) {
            return 0;
        }
        
        // If bonusEligibleAmount > staked amount, cap it at staked amount
        uint256 effectiveBonusAmount = _bonusEligibleAmount > _amount ? _amount : _bonusEligibleAmount;
        
        // Calculate base rewards (apply to all staked tokens)
        uint256 baseRewards = _amount.mul(_baseAPR).mul(_timeElapsed).div(10000).div(365 days);
        
        // Calculate bonus rewards (apply only to eligible amount)
        uint256 bonusRewards = effectiveBonusAmount.mul(_bonusAPR).mul(_timeElapsed).div(10000).div(365 days);
        
        return baseRewards.add(bonusRewards);
    }
    
    /**
     * @dev Calculate pending rewards across all APR periods
     * @param _user Address of the staker
     * @return Pending rewards amount
     */
    function calculatePendingRewards(address _user) public view returns (uint256) {
        StakingInfo memory info = stakingInfo[_user];
        if (info.amount == 0) {
            return info.unclaimedRewards;
        }
        
        // Start with unclaimed rewards from previous periods
        uint256 totalRewards = info.unclaimedRewards;
        
        // Get NFT count and calculate bonus eligible amount
        uint256 nftCount = IERC721(nftContract).balanceOf(_user);
        uint256 bonusEligibleAmount = nftCount.mul(NFT_BONUS_THRESHOLD);
        
        // Keep track of the last claim time for calculations
        uint256 periodStartTime = info.lastClaimTime;
        uint256 periodEndTime;
        uint256 timeElapsed;
        uint256 currentBaseAPR;
        uint256 currentBonusAPR;
        
        // Calculate rewards for each APR period since last claim
        for (uint256 i = 0; i < aprHistory.length; i++) {
            // Skip snapshots from before the last claim
            if (aprHistory[i].timestamp <= periodStartTime && i < aprHistory.length - 1) {
                continue;
            }
            
            // Set current APRs based on this snapshot
            currentBaseAPR = aprHistory[i].baseAPR;
            currentBonusAPR = aprHistory[i].bonusAPR;
            
            // Calculate period end time
            if (i == aprHistory.length - 1) {
                // Last snapshot - period ends now
                periodEndTime = block.timestamp;
            } else if (aprHistory[i + 1].timestamp > periodStartTime) {
                // Next snapshot is after period start - period ends at next snapshot
                periodEndTime = aprHistory[i + 1].timestamp;
            } else {
                // Next snapshot is before or at period start - skip this snapshot
                continue;
            }
            
            // Calculate elapsed time in this period
            timeElapsed = periodEndTime.sub(periodStartTime);
            
            // Add rewards for this period
            if (timeElapsed > 0) {
                totalRewards = totalRewards.add(calculatePeriodRewards(
                    info.amount,
                    timeElapsed,
                    currentBaseAPR,
                    currentBonusAPR,
                    bonusEligibleAmount
                ));
                
                // Move to next period
                periodStartTime = periodEndTime;
            }
            
            // If we've reached the current time, stop
            if (periodEndTime >= block.timestamp) {
                break;
            }
        }
        
        return totalRewards;
    }
    
    /**
     * @dev Update unclaimed rewards for a user
     * @param _user Address of the staker
     */
    function _updateUnclaimedRewards(address _user) internal {
        uint256 pendingRewards = calculatePendingRewards(_user);
        if (pendingRewards > 0) {
            stakingInfo[_user].unclaimedRewards = pendingRewards;
        }
        stakingInfo[_user].lastClaimTime = block.timestamp;
    }
    
    /**
     * @dev Internal function to claim rewards
     * @param _user Address of the staker
     */
    function _claimRewards(address _user) internal {
        uint256 pendingRewards = calculatePendingRewards(_user);
        if (pendingRewards == 0) {
            return;
        }
        
        // Update staking info
        stakingInfo[_user].lastClaimTime = block.timestamp;
        stakingInfo[_user].unclaimedRewards = 0;
        stakingInfo[_user].totalClaimed = stakingInfo[_user].totalClaimed.add(pendingRewards);
        
        // Transfer rewards to user
        IERC20(grandPepeToken).safeTransfer(_user, pendingRewards);
        
        emit RewardsClaimed(_user, pendingRewards);
    }
    
    /**
     * @dev Claim pending rewards (public facing function)

    function claimRewards() external nonReentrant {
        uint256 pendingRewards = calculatePendingRewards(msg.sender);
        require(pendingRewards > 0, "No rewards to claim");
        
        _claimRewards(msg.sender);
    }

     * @dev Compound pending rewards by adding them to the staked amount

    function compoundRewards() external nonReentrant {
        uint256 pendingRewards = calculatePendingRewards(msg.sender);

    /**
     * @dev Get current APR for a specific user (includes both base and applicable bonus)
     * @param _user Address of the staker
     * @return User's current effective APR in basis points
     */
    function getUserAPR(address _user) external view returns (uint256) {
        StakingInfo memory info = stakingInfo[_user];
        if (info.amount == 0) {
            return 0;
        }
        
        uint256 nftCount = IERC721(nftContract).balanceOf(_user);
        uint256 bonusEligibleAmount = nftCount.mul(NFT_BONUS_THRESHOLD);
        
        // If bonusEligibleAmount > staked amount, cap it at staked amount
        bonusEligibleAmount = bonusEligibleAmount > info.amount ? info.amount : bonusEligibleAmount;
        
        // Get the most recent APR snapshot
        APRSnapshot memory latestAPR = aprHistory[aprHistory.length - 1];
        
        // If entire balance is eligible for bonus
        if (bonusEligibleAmount >= info.amount) {
            return latestAPR.baseAPR.add(latestAPR.bonusAPR);
        }
        
        // Calculate weighted average APR
        uint256 weightedBonusAPR = bonusEligibleAmount.mul(latestAPR.bonusAPR).div(info.amount);
        uint256 weightedBaseAPR = info.amount.mul(latestAPR.baseAPR).div(info.amount);
        
        return weightedBaseAPR.add(weightedBonusAPR);
    }
