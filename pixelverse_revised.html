Audit Methodology

This security audit was conducted using a combination of automated and manual analysis techniques to identify vulnerabilities and security issues in the provided code.

Our approach included:

  • Static Analysis: Automated scanning tools were used to identify common vulnerabilities, coding errors, and security anti-patterns.
  • Manual Code Review: Experienced security auditors performed a line-by-line review of the codebase to identify logical flaws, access control issues, and other vulnerabilities that automated tools might miss.
  • Architecture Analysis: The overall system design was evaluated for security implications, focusing on the interaction between components and potential attack vectors.
  • Gas Optimization: For smart contracts, we analyzed gas usage patterns and identified opportunities for optimization.
  • Best Practices Review: The code was evaluated against industry best practices and standards for secure development.

Severity Classification:

  • High: Vulnerabilities that can lead to asset loss, significant data compromise, or other severe impacts.
  • Medium: Vulnerabilities that could potentially lead to asset loss under specific circumstances or represent significant flaws in the system's design.
  • Low: Vulnerabilities that are limited in scope or require unusual circumstances to exploit.
  • Informational: Issues that do not pose an immediate security risk but should be addressed as best practice.

Key Findings

H-01

Reentrancy Vulnerability

The `buyListedPlot` and `buyListedBundle` functions are vulnerable to reentrancy attacks. The USDC token transfer is executed after the internal state updates, which could allow an attacker to repeatedly call these functions before the state is updated, draining the contract's USDC balance.
REVISED SEVERITY
Original: High Revised: Low
The contract does not hold any USDC.

Lines 342-384 and 718-778
IMPACT
An attacker could repeatedly call the `buyListedPlot` or `buyListedBundle` functions to drain the contract's USDC balance.
RECOMMENDATION
Implement the Checks-Effects-Interactions pattern to mitigate reentrancy attacks. Perform all state updates before executing external calls.

    uint256 public constant GRID_SIZE = 100;
    uint256 public constant MAX_PLOTS = GRID_SIZE * GRID_SIZE; // 10,000 plots

    /**
     * @dev Converts x,y coordinates to a plot ID
     */
    function coordinatesToPlotId(uint8 x, uint8 y) public pure returns (uint256) {
        require(x < GRID_SIZE, "X coordinate out of bounds");
        require(y < GRID_SIZE, "Y coordinate out of bounds");
        return (y * GRID_SIZE) + x;
    }
    
    /**
     * @dev Converts a plot ID to x,y coordinates
     */
    function plotIdToCoordinates(uint256 plotId) public pure returns (uint8 x, uint8 y) {
        require(plotId < MAX_PLOTS, "Plot ID out of bounds");
        y = uint8(plotId / GRID_SIZE);
        x = uint8(plotId % GRID_SIZE);
        return (x, y);
    }

    /**
     * @dev Calculates and transfers fee amount split among the four fee receivers
     * @return totalFeeAmount The total fee amount that was distributed
     */
    function _processFee(uint256 amount, uint256 feePercent, string memory feeType) internal returns (uint256) {
        uint256 totalFeeAmount = (amount * feePercent) / 10000;
        if (totalFeeAmount == 0) {
            return 0;
        }
        
        // Calculate individual fee amounts
        uint256[] memory feeAmounts = new uint256[](4);
        uint256 remainingFee = totalFeeAmount;
        
        // Split fees into 4 equal parts (handling remainder)
        for (uint256 i = 0; i < 4; i++) {
            if (i == 3) {
                // Last receiver gets any remainder
                feeAmounts[i] = remainingFee;
            } else {
                feeAmounts[i] = totalFeeAmount / 4;
                remainingFee -= feeAmounts[i];
            }
            
            // Transfer fee to receiver if amount is non-zero
            if (feeAmounts[i] > 0 && feeReceivers[i] != address(0)) {
                require(
                    usdcToken.transferFrom(msg.sender, feeReceivers[i], feeAmounts[i]),
                    "Fee transfer failed"
                );
            }
        }
        
        emit FeesCollected(feeReceivers, feeAmounts, feeType);
        return totalFeeAmount;
    }
    
    /**
     * @dev Purchases a plot at the given coordinates using USDC
     * @notice Protected against reentrancy attacks
     * @notice Requires USDC approval before calling
     */
    function purchasePlot(uint8 x, uint8 y) external nonReentrant {
        uint256 plotId = coordinatesToPlotId(x, y);
        
        // Check if the plot is already owned
        require(plots[plotId].owner == address(0), "Plot already owned");
        
        // Process platform fee
        _processFee(plotPrice, platformFeePercent, "PRIMARY_SALE");
        
        // Transfer USDC tokens
        require(
            usdcToken.transferFrom(msg.sender, address(this), plotPrice - (plotPrice * platformFeePercent / 10000)),
            "USDC transfer failed"
        );
        
        // Store plot details
        plots[plotId] = Plot({
            owner: msg.sender,
            x: x,
            y: y,
            isForSale: false,
            salePrice: 0,
            purchaseTimestamp: block.timestamp
        });
        
        // Update ownership tracking
        _plotsOwnedByUser[msg.sender].push(plotId);
        _plotIndexInOwnerArray[msg.sender][plotId] = _plotsOwnedByUser[msg.sender].length - 1;
        
        // Update stats
        totalPlotsSold++;
        totalVolume += plotPrice;
        
        emit PlotPurchased(msg.sender, plotId, x, y, plotPrice);
    }
    
    /**
     * @dev Lists a plot for sale
     */
    function listPlotForSale(uint256 plotId, uint256 price) external {
        require(plots[plotId].owner == msg.sender, "Not owner of plot");
        require(price > 0, "Price must be greater than zero");
        
        // Check if plot is part of any bundle that's listed for sale
        bool isBundlePlot = false;
        uint256[] memory ownerBundles = _bundlesOwnedByUser[msg.sender];
        for (uint256 i = 0; i < ownerBundles.length; i++) {
            PlotBundle storage bundle = plotBundles[ownerBundles[i]];
            if (bundle.isForSale) {
                for (uint256 j = 0; j < bundle.plotIds.length; j++) {
                    if (bundle.plotIds[j] == plotId) {
                        isBundlePlot = true;
                        break;
                    }
                }
                if (isBundlePlot) break;
            }
        }
        require(!isBundlePlot, "Plot is part of a bundle listed for sale");
        
        plots[plotId].isForSale = true;
        plots[plotId].salePrice = price;
        
        // Add to plots for sale array if not already listed
        if (_plotIndexInSaleArray[plotId] == 0 && _plotsForSale.length == 0) {
            _plotsForSale.push(plotId);
            _plotIndexInSaleArray[plotId] = _plotsForSale.length - 1;
        } else if (_plotsForSale.length > 0 && _plotsForSale[_plotIndexInSaleArray[plotId]] != plotId) {
            _plotsForSale.push(plotId);
            _plotIndexInSaleArray[plotId] = _plotsForSale.length - 1;
        }
        
        emit PlotListed(msg.sender, plotId, price);
    }
    
    /**
     * @dev Removes a plot from sale
     */
    function unlistPlot(uint256 plotId) external {
        require(plots[plotId].owner == msg.sender, "Not owner of plot");
        require(plots[plotId].isForSale, "Plot is not for sale");
        
        plots[plotId].isForSale = false;
        plots[plotId].salePrice = 0;
        
        // Remove from plots for sale array
        uint256 lastPlotIndex = _plotsForSale.length - 1;
        uint256 plotIndex = _plotIndexInSaleArray[plotId];
        
        if (plotIndex != lastPlotIndex) {
            uint256 lastPlotId = _plotsForSale[lastPlotIndex];
            _plotsForSale[plotIndex] = lastPlotId;
            _plotIndexInSaleArray[lastPlotId] = plotIndex;
        }
        
        _plotsForSale.pop();
        delete _plotIndexInSaleArray[plotId];
        
        emit PlotUnlisted(msg.sender, plotId);
    }
    
    /**
     * @dev Purchases a plot that is listed for sale by another user using USDC
     * @notice Protected against reentrancy attacks
     * @notice Requires USDC approval before calling
     */
    function buyListedPlot(uint256 plotId) external nonReentrant {
        require(plots[plotId].isForSale, "Plot is not for sale");
        require(plots[plotId].owner != msg.sender, "You already own this plot");
        
        address payable seller = payable(plots[plotId].owner);
        uint256 salePrice = plots[plotId].salePrice;
        
        // Process platform fee and royalty
        uint256 platformFeeAmount = _processFee(salePrice, platformFeePercent, "SECONDARY_SALE");
        uint256 royaltyAmount = _processFee(salePrice, resaleRoyaltyPercent, "RESALE_ROYALTY");
        uint256 sellerAmount = salePrice - platformFeeAmount - royaltyAmount;
        
        // Transfer USDC to seller
        if (sellerAmount > 0) {
            require(
                usdcToken.transferFrom(msg.sender, seller, sellerAmount),
                "USDC transfer to seller failed"
            );
        }
        
        // Remove plot from seller's ownership
        uint256 lastPlotIndex = _plotsOwnedByUser[seller].length - 1;
        uint256 plotIndex = _plotIndexInOwnerArray[seller][plotId];
        
        if (plotIndex != lastPlotIndex) {
            uint256 lastPlotId = _plotsOwnedByUser[seller][lastPlotIndex];
            _plotsOwnedByUser[seller][plotIndex] = lastPlotId;
            _plotIndexInOwnerArray[seller][lastPlotId] = plotIndex;
        }
        
        _plotsOwnedByUser[seller].pop();
        delete _plotIndexInOwnerArray[seller][plotId];
        
        // Remove from plots for sale array
        lastPlotIndex = _plotsForSale.length - 1;
        plotIndex = _plotIndexInSaleArray[plotId];
        
        if (plotIndex != lastPlotIndex) {
            uint256 lastPlotId = _plotsForSale[lastPlotIndex];
            _plotsForSale[plotIndex] = lastPlotId;
            _plotIndexInSaleArray[lastPlotId] = plotIndex;
        }
        
        _plotsForSale.pop();
        delete _plotIndexInSaleArray[plotId];
        
        // Update plot details
        plots[plotId].owner = msg.sender;
        plots[plotId].isForSale = false;
        plots[plotId].salePrice = 0;
        plots[plotId].purchaseTimestamp = block.timestamp;
        
        // Add plot to buyer's ownership
        _plotsOwnedByUser[msg.sender].push(plotId);
        _plotIndexInOwnerArray[msg.sender][plotId] = _plotsOwnedByUser[msg.sender].length - 1;
        
        // Update stats
        totalResaleTransactions++;
        totalVolume += salePrice;
        
        emit PlotSold(seller, msg.sender, plotId, salePrice);
    }
    
    /**
     * @dev Purchases multiple plots at once using USDC
     * @notice Protected against reentrancy attacks
     * @notice Requires USDC approval before calling
     */
    function purchaseMultiplePlots(uint8[] calldata x, uint8[] calldata y) external nonReentrant {
        require(x.length == y.length, "Coordinate arrays must have the same length");
        require(x.length > 0, "Must purchase at least one plot");
        
        uint256 totalCost = plotPrice * x.length;
        
        // Process platform fee
        uint256 feeAmount = _processFee(totalCost, platformFeePercent, "PRIMARY_SALE_BULK");
        
        // Transfer USDC tokens
        require(
            usdcToken.transferFrom(msg.sender, address(this), totalCost - feeAmount),
            "USDC transfer failed"
        );
        
        uint256[] memory purchasedPlotIds = new uint256[](x.length);
        
        for (uint256 i = 0; i < x.length; i++) {
            uint256 plotId = coordinatesToPlotId(x[i], y[i]);
            
            // Check if the plot is already owned
            require(plots[plotId].owner == address(0), "Plot already owned");
            
            // Store plot details
            plots[plotId] = Plot({
                owner: msg.sender,
                x: x[i],
                y: y[i],
                isForSale: false,
                salePrice: 0,
                purchaseTimestamp: block.timestamp
            });
            
            // Update ownership tracking
            _plotsOwnedByUser[msg.sender].push(plotId);
            _plotIndexInOwnerArray[msg.sender][plotId] = _plotsOwnedByUser[msg.sender].length - 1;
            
            // Store plot ID for event
            purchasedPlotIds[i] = plotId;
        }
        
        // Update stats
        totalPlotsSold += x.length;
        totalVolume += totalCost;
        
        emit MultiplePlotsPurchased(msg.sender, purchasedPlotIds, totalCost);
    }
    
    /**
     * @dev Creates a bundle of plots that can be listed for sale together
     * @param plotIds Array of plot IDs to include in the bundle
     */
    function createPlotBundle(uint256[] calldata plotIds) external {
        require(plotIds.length > 1, "Bundle must include at least 2 plots");
        
        // Verify ownership of all plots and check for duplicates with O(n) complexity
        for (uint256 i = 0; i < plotIds.length; i++) {
            uint256 plotId = plotIds[i];
            require(plotId < MAX_PLOTS, "Plot ID out of bounds");
            require(plots[plotId].owner == msg.sender, "Not owner of all plots");
            require(!plots[plotId].isForSale, "Plot already listed for individual sale");
            
            // Check for duplicates
            for (uint256 j = 0; j < i; j++) {
                require(plotIds[j] != plotId, "Duplicate plot IDs not allowed");
            }
        }
        
        // Create new bundle
        uint256 bundleId = _nextBundleId++;
        
        // Create a copy of plotIds to store in the bundle
        uint256[] memory bundlePlotIds = new uint256[](plotIds.length);
        for (uint256 i = 0; i < plotIds.length; i++) {
            bundlePlotIds[i] = plotIds[i];
            // Mark plots as part of this bundle
            _plotInBundle[bundleId][plotIds[i]] = true;
        }
        
        plotBundles[bundleId] = PlotBundle({
            plotIds: bundlePlotIds,
            owner: msg.sender,
            isForSale: false,
            salePrice: 0,
            creationTimestamp: block.timestamp
        });
        
        // Update ownership tracking
        _bundlesOwnedByUser[msg.sender].push(bundleId);
        _bundleIndexInOwnerArray[msg.sender][bundleId] = _bundlesOwnedByUser[msg.sender].length - 1;
        
        emit PlotBundleCreated(msg.sender, bundleId, plotIds);
    }
    
    /**
     * @dev Dismantles (unbundles) a plot bundle
     * @param bundleId The ID of the bundle to dismantle
     */
    function unbundlePlots(uint256 bundleId) external {
        require(plotBundles[bundleId].owner == msg.sender, "Not owner of bundle");
        require(!plotBundles[bundleId].isForSale, "Cannot unbundle a bundle listed for sale");
        
        PlotBundle storage bundle = plotBundles[bundleId];
        uint256[] memory plotIds = bundle.plotIds;
        
        // Remove bundle from owner's bundles
        uint256 lastBundleIndex = _bundlesOwnedByUser[msg.sender].length - 1;
        uint256 bundleIndex = _bundleIndexInOwnerArray[msg.sender][bundleId];
        
        if (bundleIndex != lastBundleIndex) {
            uint256 lastBundleId = _bundlesOwnedByUser[msg.sender][lastBundleIndex];
            _bundlesOwnedByUser[msg.sender][bundleIndex] = lastBundleId;
            _bundleIndexInOwnerArray[msg.sender][lastBundleId] = bundleIndex;
        }
        
        _bundlesOwnedByUser[msg.sender].pop();
        delete _bundleIndexInOwnerArray[msg.sender][bundleId];
        
        // Remove plots from bundle mapping
        for (uint256 i = 0; i < plotIds.length; i++) {
            delete _plotInBundle[bundleId][plotIds[i]];
        }
        
        // Emit event before deleting the bundle data
        emit PlotBundleDismantled(msg.sender, bundleId, plotIds);
        
        // Clear bundle data
        delete plotBundles[bundleId];
    }
    
    /**
     * @dev Lists a bundle of plots for sale
     */
    function listBundleForSale(uint256 bundleId, uint256 price) external {
        require(plotBundles[bundleId].owner == msg.sender, "Not owner of bundle");
        require(price > 0, "Price must be greater than zero");
        
        PlotBundle storage bundle = plotBundles[bundleId];
        
        // Verify ownership of all plots still belongs to the caller
        for (uint256 i = 0; i < bundle.plotIds.length; i++) {
            uint256 plotId = bundle.plotIds[i];
            require(plots[plotId].owner == msg.sender, "No longer owner of all plots");
            require(!plots[plotId].isForSale, "Plot already listed for individual sale");
        }
        
        // List the bundle for sale
        bundle.isForSale = true;
        bundle.salePrice = price;
        
        // Add to bundles for sale array
        _bundlesForSale.push(bundleId);
        _bundleIndexInSaleArray[bundleId] = _bundlesForSale.length - 1;
        
        emit PlotBundleListed(msg.sender, bundleId, price);
    }
    
    /**
     * @dev Removes a bundle from sale
     */
    function unlistBundle(uint256 bundleId) external {
        require(plotBundles[bundleId].owner == msg.sender, "Not owner of bundle");
        require(plotBundles[bundleId].isForSale, "Bundle is not for sale");
        
        PlotBundle storage bundle = plotBundles[bundleId];
        bundle.isForSale = false;
        bundle.salePrice = 0;
        
        // Remove from bundles for sale array
        uint256 lastBundleIndex = _bundlesForSale.length - 1;
        uint256 bundleIndex = _bundleIndexInSaleArray[bundleId];
        
        if (bundleIndex != lastBundleIndex) {
            uint256 lastBundleId = _bundlesForSale[lastBundleIndex];
            _bundlesForSale[bundleIndex] = lastBundleId;
            _bundleIndexInSaleArray[lastBundleId] = bundleIndex;
        }
        
        _bundlesForSale.pop();
        delete _bundleIndexInSaleArray[bundleId];
        
        emit PlotBundleUnlisted(msg.sender, bundleId);
    }
    
    /**
     * @dev Purchases a bundle of plots that is listed for sale using USDC
     * @notice Protected against reentrancy attacks
     * @notice Requires USDC approval before calling
     */
    function buyListedBundle(uint256 bundleId) external nonReentrant {
        PlotBundle storage bundle = plotBundles[bundleId];
        
        require(bundle.isForSale, "Bundle is not for sale");
        require(bundle.owner != msg.sender, "You already own this bundle");
        
        address payable seller = payable(bundle.owner);
        uint256 salePrice = bundle.salePrice;
        
        // Process platform fee and royalty
        uint256 platformFeeAmount = _processFee(salePrice, platformFeePercent, "SECONDARY_SALE_BUNDLE");
        uint256 royaltyAmount = _processFee(salePrice, resaleRoyaltyPercent, "RESALE_ROYALTY_BUNDLE");
        uint256 sellerAmount = salePrice - platformFeeAmount - royaltyAmount;
        
        // Transfer USDC to seller
        if (sellerAmount > 0) {
            require(
                usdcToken.transferFrom(msg.sender, seller, sellerAmount),
                "USDC transfer to seller failed"
            );
        }
        
        // Transfer ownership of all plots in the bundle
        for (uint256 i = 0; i < bundle.plotIds.length; i++) {
            uint256 plotId = bundle.plotIds[i];
            
            // Update plot ownership
            plots[plotId].owner = msg.sender;
            plots[plotId].purchaseTimestamp = block.timestamp;
            
            // Remove plot from seller's ownership
            uint256 lastPlotIndex = _plotsOwnedByUser[seller].length - 1;
            uint256 plotIndex = _plotIndexInOwnerArray[seller][plotId];
            
            if (plotIndex != lastPlotIndex) {
                uint256 lastPlotId = _plotsOwnedByUser[seller][lastPlotIndex];
                _plotsOwnedByUser[seller][plotIndex] = lastPlotId;
                _plotIndexInOwnerArray[seller][lastPlotId] = plotIndex;
            }
            
            _plotsOwnedByUser[seller].pop();
            delete _plotIndexInOwnerArray[seller][plotId];
            
            // Add plot to buyer's ownership
            _plotsOwnedByUser[msg.sender].push(plotId);
            _plotIndexInOwnerArray[msg.sender][plotId] = _plotsOwnedByUser[msg.sender].length - 1;
        }
        
        // Remove bundle from seller's ownership
        uint256 lastBundleIndex = _bundlesOwnedByUser[seller].length - 1;
        uint256 bundleIndex = _bundleIndexInOwnerArray[seller][bundleId];
        
        if (bundleIndex != lastBundleIndex) {
            uint256 lastBundleId = _bundlesOwnedByUser[seller][lastBundleIndex];
            _bundlesOwnedByUser[seller][bundleIndex] = lastBundleId;
            _bundleIndexInOwnerArray[seller][lastBundleId] = bundleIndex;
        }
        
        _bundlesOwnedByUser[seller].pop();
        delete _bundleIndexInOwnerArray[seller][bundleId];
        
        // Remove from bundles for sale array
        lastBundleIndex = _bundlesForSale.length - 1;
        bundleIndex = _bundleIndexInSaleArray[bundleId];
        
        if (bundleIndex != lastBundleIndex) {
            uint256 lastBundleId = _bundlesForSale[lastBundleIndex];
            _bundlesForSale[bundleIndex] = lastBundleId;
            _bundleIndexInSaleArray[lastBundleId] = bundleIndex;
        }
        
        _bundlesForSale.pop();
        delete _bundleIndexInSaleArray[bundleId];
        
        // Update bundle details
        bundle.owner = msg.sender;
        bundle.isForSale = false;
        bundle.salePrice = 0;
        
        // Add bundle to buyer's ownership
        _bundlesOwnedByUser[msg.sender].push(bundleId);
        _bundleIndexInOwnerArray[msg.sender][bundleId] = _bundlesOwnedByUser[msg.sender].length - 1;
        
        // Update stats
        totalResaleTransactions++;
        totalBundlesSold++;
        totalVolume += salePrice;
        
        emit PlotBundleSold(seller, msg.sender, bundleId, salePrice);
    }

    function getAllPlots(uint256 start, uint256 limit) external view returns (
        uint256[] memory plotIds,
        address[] memory owners,
        uint256 ownershipCount
    ) {
        // Count total owned plots first
        uint256 totalOwned = 0;
        for (uint256 i = 0; i < MAX_PLOTS; i++) {
            if (plots[i].owner != address(0)) {
                totalOwned++;
            }
        }
        
        // Handle empty case
        if (totalOwned == 0) {
            return (new uint256[](0), new address[](0), 0);
        }
        
        // Validate pagination parameters
        if (start >= totalOwned) {
            return (new uint256[](0), new address[](0), totalOwned);
        }
        
        // Calculate how many results to return based on limit
        uint256 resultCount = (start + limit > totalOwned) ? (totalOwned - start) : limit;
        
        // Initialize return arrays
        uint256[] memory resultPlotIds = new uint256[](resultCount);
        address[] memory resultOwners = new address[](resultCount);
        
        // Fill return arrays
        uint256 currentIndex = 0;
        uint256 foundCount = 0;
        
        for (uint256 i = 0; i < MAX_PLOTS && currentIndex < resultCount; i++) {
            if (plots[i].owner != address(0)) {
                if (foundCount >= start) {
                    resultPlotIds[currentIndex] = i;
                    resultOwners[currentIndex] = plots[i].owner;
                    currentIndex++;
                }
                foundCount++;
            }
        }
        
        return (resultPlotIds, resultOwners, totalOwned);
    }
    
    /**
     * @dev Gets the plots surrounding a specific plot
     * @param plotId The plot ID to get neighbors for
     * @return An array of up to 8 neighboring plot IDs (fewer if on edge)
     */
    function getNeighboringPlots(uint256 plotId) external pure returns (uint256[] memory) {
        require(plotId < MAX_PLOTS, "Plot ID out of bounds");
        
        (uint8 x, uint8 y) = plotIdToCoordinates(plotId);
        uint256[] memory neighbors = new uint256[](8);
        uint256 count = 0;
        
        // Check top-left
        if (x > 0 && y > 0) {
            neighbors[count++] = coordinatesToPlotId(x - 1, y - 1);
        }
        
        // Check top
        if (y > 0) {
            neighbors[count++] = coordinatesToPlotId(x, y - 1);
        }
        
        // Check top-right
        if (x < GRID_SIZE - 1 && y > 0) {
            neighbors[count++] = coordinatesToPlotId(x + 1, y - 1);
        }
        
        // Check left
        if (x > 0) {
            neighbors[count++] = coordinatesToPlotId(x - 1, y);
        }
        
        // Check right
        if (x < GRID_SIZE - 1) {
            neighbors[count++] = coordinatesToPlotId(x + 1, y);
        }
        
        // Check bottom-left
        if (x > 0 && y < GRID_SIZE - 1) {
            neighbors[count++] = coordinatesToPlotId(x - 1, y + 1);
        }
        
        // Check bottom
        if (y < GRID_SIZE - 1) {
            neighbors[count++] = coordinatesToPlotId(x, y + 1);
        }
        
        // Check bottom-right
        if (x < GRID_SIZE - 1 && y < GRID_SIZE - 1) {
            neighbors[count++] = coordinatesToPlotId(x + 1, y + 1);
        }
        
        // Resize array to actual number of neighbors
        assembly {
            mstore(neighbors, count)
        }
        
        return neighbors;
    }
